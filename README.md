## How to open the project

1. افتح برنامج visual studio code
2. ثبت الاضافة "Live Server"
3. افتح الملف "index.html"
4. بتلقى في اسفل اليمين زر اسمه "Go Live" اضغط عليه وبيفتح لك الموقع
