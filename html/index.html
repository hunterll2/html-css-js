Learn HTML

HTML is the language used to create the basic strcutrue of the any web page.

HTML Elements

HTML elements consist of several parts, including the opening and closing tags, the content, and the attributes.

  • The opening tag: This consists of the element name, wrapped in angle brackets. It indicates the start of the element and the point at which the element's effects begin.
  • The closing tag: This is the same as the opening tag, but with a forward slash before the element name. It indicates the end of the element and the point at which the element's effects stop.
  • The content: This is the content of the element, which can be text, other elements, or a combination of both.
  • The attributes: HTML elements can have attributes, which provide additional information about the element. They are specified in the opening tag of the element and take the form of name-value pairs.
  • The element: The opening tag, the closing tag, and the content together make up the element.

HTML Forms

An HTML form is used to collect user input. The user input is most often sent to a server for processing.

The <form> Element

A container for different types of input elements, such as: text fields, checkboxes, radio buttons, submit buttons, etc.

<form> - Attributes

  • action="/"

    Defines the action to be performed when the form is submitted.

  • target="_selft|_blank"

    Where to display the response that is received after submitting the form.

  • method="get|post"

    The HTTP method to be used when submitting the form data. The form-data can be sent as URL variables (with method="get") or as HTTP post transaction (with method="post").

  • autocomplete="on|off"
  • novalidate

<form> - Elements

<input> - Attributes

  • type
    • text
    • email
    • password
    • number
    • date
    • time
    • radio
    • checkbox
    • range
    • file
    • submit
    • reset
    • button
  • id

    The id can be used by the label element to reference the input element.

  • name

    The name used to reference an input element when the form data is submitted.

    It also used to taied together multiple radio/checkbox elements into one group.

  • value

    Specifies the initial value of an input.

  • max (For: number, range, date, datetime-local, month, time, week)

    Specifies the maximum value for an input field.

  • min (For: number, range, date, datetime-local, month, time, week)

    Specifies the minimum value for an input field.

  • step (For: number, range, date, datetime-local, month, time, week)

    Specifies the legal number intervals for an input field

  • size (For: text, email, password, search, tel, url)

    Specifies the width (in characters) of an input field

  • minlength

    Specifies the minimum number of character for an input field.

  • maxlength

    Specifies the maximum number of character for an input field.

  • placeholder
  • list

    Refers to a datalist element that contains pre-defined options for an input element.

  • required
  • readonly
  • disabled

    The value of input field will not be sent when submitting the form!

  • checked (For: checkbox, radio)
  • autofocus
  • autocomplete (For: text, search, url, tel, email, password, datepickers, range, color)
  • multiple (For: email, file)
  • accept (For: file)

    Specifies a filter for what file types the user can pick from the file input dialog box

    Tip: Do not use this attribute as a validation tool. File uploads should be validated on the server.
